Banking Management System

Operations:
1. Existing user can login, new user can sign up for a new account.
2. Users can view their account details.
3. Users can withdraw and deposit money in their account.
4. Users can transfer money to another user.

Tools used: HTML, CSS, PHP, MYSQL

Instructions:

1. Create a new database, name it "bank".
2. Create a new table in the database, name it "customers".
3. Open the folder on a server.
